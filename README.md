## 👩‍💻 Project Overview

This project is a **simple Registration Form** built using **HTML and JavaScript** that performs **client-side form validation**.

The form validates user input before submission to ensure correct and complete data entry.

---

## ✨ Features

✅ Name field validation
✅ Email format verification using Regex
✅ Empty field checking
✅ Alert messages for invalid inputs
✅ Successful submission confirmation
✅ Clean and beginner-friendly UI

---

## 📋 Form Fields

* 👤 Name
* 📧 Email
* 🚻 Gender Selection
* 🎂 Date of Birth

---

## 🧠 Validation Logic

The JavaScript function `validateForm()` checks:

1. Name field should not be empty.
2. Email field should not be empty.
3. Email must match valid email format using **Regular Expression**.
4. Shows success alert when all validations pass.
